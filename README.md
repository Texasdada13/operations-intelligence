# Operations Intelligence

AI-powered operational excellence platform - Your Fractional COO.

## Features

- **Process Analytics**: Analyze process efficiency, identify bottlenecks, optimize cycle times
- **KPI Benchmarking**: Compare operational metrics against industry standards
- **Capacity Planning**: Forecast demand, allocate resources, plan capacity
- **Risk Assessment**: Multi-dimensional operational risk classification
- **AI Consultation**: Claude-powered COO consultant for instant advice

## Tech Stack

- **Backend**: Flask, SQLAlchemy, PostgreSQL
- **AI**: Anthropic Claude API
- **Frontend**: Bootstrap 5, Vanilla JS
- **Deployment**: Render

## Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Set environment variables
export ANTHROPIC_API_KEY=your-key-here
export SECRET_KEY=your-secret-key

# Run locally
python web/app.py
```

## Deployment

Deploy to Render using the included `render.yaml` blueprint.
